# sreekar.coffee

A personal coffee blog documenting AeroPress brewing experiments — recipes, tasting notes, café visits, and coffee rabbit holes.

## Stack

- Vanilla HTML + CSS (no frameworks)
- [Umami](https://umami.is) for privacy-friendly analytics
- Static hosting

## Structure

```
├── index.html          # Home
├── styles.css          # All styles (single shared sheet)
├── rss.xml             # RSS feed (auto-generated)
├── generate_rss.py     # Script to regenerate rss.xml
├── images/
├── .github/
│   └── workflows/
│       └── update-rss.yml  # Auto-updates rss.xml on blog push
└── blog/
    ├── index.html      # Post listing
    └── cup-*.html      # Brewing logs
```

## Design

Coffee journal aesthetic — warm cream background (`#F8F2E8`), full-page dot grid, handwritten fonts (Caveat + Kalam), minimal layout. No build step, no dependencies.

## RSS

Feed lives at `/rss.xml`, linked in the homepage footer. Auto-regenerated by GitHub Actions whenever a file in `blog/` is pushed. To regenerate manually:

```bash
python3 generate_rss.py
```

## Local dev

```bash
python3 -m http.server 3000
```
